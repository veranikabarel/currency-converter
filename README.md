## General info
A small web based currency converter that supposed to be a coding assignment for recruiting purposes. This converter should be able to  answer the following questions:
*What is 200 SEK converted into multiple other currencies like USD, GBP and SGD?
*How much has currency changed in valuation between March 26th, 2015 and June 13th, 2017?

## Installation and running
##### Clone the repo and cd into the directory
```bash
git clone https://github.com/veranikabarel/currency-converter.git
```

##### Install All dependent modules if you do not have already
```bash
npm install
```

##### To start the React packager
```bash
npm start
```
