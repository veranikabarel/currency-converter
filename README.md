## Currency Converter 

This is a coding assignment, a small web based currency converter that can answer the questions:
What is 200 SEK converted into multiple other currencies like USD, GBP and SGD?
How much has currency changed in valuation between March 26th, 2015 and June 13th, 2017?

### Installation and Running
##### Clone the repo and cd into the directory
```bash
git clone https://github.com/ashwinpreet/ReactNativeExamples.git
```

##### Install All dependent modules if you do not have already
```bash
npm install
```

##### To start the React packager
```bash
npm start
```
